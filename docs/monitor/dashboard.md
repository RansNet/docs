# Monitoring Dashboard

## What This Does
The Monitoring Dashboard provides a summarized, real-time view of the status and performance of devices within the selected entity.

## Why It Matters
It allows administrators to quickly assess network health, identify issues, and monitor key performance metrics from a single centralized view.

## Where to Access
Navigate to **ORCHESTRATOR → Monitoring → Dashboard**, then select the desired entity using the **[Entity]** button in the top-right corner.

---

## Overview

The dashboard layout is fully customizable. You can:

- Choose which sections to display  
- Select specific devices (hosts) to monitor  
- Define which monitoring items and graphs are shown for each device  

---

## Customizing the Dashboard

### Step 1 — Select Sections

Click the **[Customize Page]** button in the top-right corner to choose which sections to display on the dashboard.

![mfusion dashboard customization](../images/monitor-dashboard-1.png)

---

### Step 2 — Select Devices and Graphs

Click the **[Customize Section]** button to configure each section:

- Select the devices (hosts) to include  
- Expand each device to choose specific monitoring items and graphs  

![mfusion dashboard customization](../images/monitor-dashboard-2.png)

---

### Step 3 — Save Changes

After saving, the dashboard will update to reflect your customized layout.

![mfusion dashboard customization](../images/monitor-dashboard-3.png)

---

## Troubleshooting

- If no data is displayed, ensure the device is online and reporting to mfusion  
- Verify that monitoring items are selected in the section configuration
